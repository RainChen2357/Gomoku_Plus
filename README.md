# Gomoku Plus · 趣味五子棋

带趣味规则的简易五子棋，支持 **Python (tkinter)** 与 **网页版** 两种运行方式。

## 玩法说明

### 基本规则

- **棋盘**：15×15 格，黑先白后，轮流在交叉点落子。
- **胜负**：任意一方在横、竖、斜任一方向**先连成五子**即获胜；棋盘下满无人连五则和棋。

### 趣味规则

| 规则 | 说明 |
|------|------|
| **落子虚影** | 移动鼠标时，在合法空位上会显示当前方棋子的半透明虚影，方便预览落子位置。 |
| **连下两颗** | 每次落子后有 **10% 概率** 触发「连下两颗」：当前方可以**再下一子**，界面顶部会提示「🎲 连下两颗！请再下一子」。 |
| **随机消失** | 每次落子后有 **10% 概率** 使棋盘上**之前已存在的某一颗棋子（黑白均可）随机消失**，刚下的子不会被消失。胜负以**落子后当前棋盘**是否连成五子为准。 |

## 运行方法

### 方式一：Python 桌面版

**环境要求**：Python 3.x，需自带 tkinter（一般已包含）。

```bash
# 克隆或下载项目后，在项目目录下执行
python gomoku.py
```

- Windows 若未装 Python，请从 [python.org](https://www.python.org/) 安装并勾选 “tcl/tk”。
- macOS / Linux 通常已带 tkinter。

### 方式二：网页版

直接用浏览器打开 `gomoku.html` 即可，无需服务器。

```bash
# macOS
open gomoku.html

# Windows
start gomoku.html

# 或在文件管理器中双击 gomoku.html
```

也可以把 `gomoku.html` 部署到 GitHub Pages 或任意静态网站，在线游玩。
